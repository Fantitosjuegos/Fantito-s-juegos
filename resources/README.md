\# App Assets



Place the following files here before running `npx @capacitor/assets generate`:



\- icon.png        — 1024x1024 PNG, no rounded corners, no transparency (required)

\- splash.png      — 2732x2732 PNG, centered logo on solid background (required)

\- icon-foreground.png  — 1024x1024 PNG, foreground layer for Android adaptive icon

\- icon-background.png  — 1024x1024 PNG, background layer for Android adaptive icon



To generate all sizes run:

&#x20; npx @capacitor/assets generate
